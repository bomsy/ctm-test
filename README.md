### Comparethemarket interview test

#### Install & Setup

```
yarn install | npm install
```

#### Build & Run
Bundles the js and css into the `/dist` folder

```
yarn build
```

Starts up the server at `http://127.0.0.1:8081/`
```
yarn start
```

#### Run Tests

```
yarn test
```
